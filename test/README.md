# 🎨 Web Image Editor - Photoshop Clone

一個功能完整的網頁版圖像編輯器，模仿 Photoshop 的核心功能，提供專業的圖像編輯體驗。

## ✨ 功能特色

### 🎨 核心繪圖工具
- **畫筆工具** - 支援壓感和不透明度調整
- **橡皮擦工具** - 精確擦除不需要的內容
- **油漆桶工具** - 智能填充相似顏色區域
- **吸管工具** - 快速拾取畫布上的顏色
- **選取工具** - 創建選取區域進行精確編輯
- **文字工具** - 添加和編輯文字內容
- **模糊工具** - 創造柔焦和景深效果
- **克隆工具** - 複製圖像區域進行修復

### 🎛️ 專業功能
- **圖層系統** - 多圖層編輯和管理
- **濾鏡效果** - 模糊、銳化、灰階、反相等
- **歷史記錄** - 支援復原/重做操作
- **顏色管理** - 前景色/背景色選擇器
- **筆刷設定** - 大小、不透明度調整
- **檔案操作** - 開啟、儲存圖像檔案

### 🤖 AI 智能功能
- **AI 去浮水印** - 智能檢測並移除圖像中的浮水印
- **AI 風格轉換** - 將照片轉換為漫畫、電影、動漫等風格
- **AI 圖像增強** - 自動提升圖像對比度和清晰度
- **AI 去背景** - 智能移除圖像背景，創建透明效果

### 🚀 用戶體驗
- **專業界面** - 仿 Photoshop 的深色主題
- **快捷鍵支援** - Ctrl+Z/Y/S/O 等常用快捷鍵
- **即時預覽** - 所有操作即時反映在畫布上
- **響應式設計** - 適配不同螢幕尺寸
- **狀態提示** - 即時顯示操作狀態和畫布資訊

## 🎮 使用方法

### 基本操作
1. 開啟 `image-editor.html` 檔案
2. 從左側工具欄選擇繪圖工具
3. 在右側面板調整工具參數
4. 在畫布上進行創作和編輯
5. 使用選單功能開啟/儲存檔案

### 工具使用
- **畫筆工具**：點擊並拖拽進行繪畫
- **橡皮擦**：擦除不需要的內容
- **油漆桶**：點擊區域進行填充
- **吸管工具**：點擊拾取顏色
- **選取工具**：拖拽創建選取區域
- **克隆工具**：Alt+點擊設定源點，然後繪製

### 快捷鍵
- **Ctrl+Z**：復原上一步操作
- **Ctrl+Y**：重做操作
- **Ctrl+S**：儲存圖像
- **Ctrl+O**：開啟檔案

### 🤖 AI 功能使用
- **去浮水印**：點擊「AI 去浮水印」按鈕，系統會自動檢測並移除浮水印
- **風格轉換**：點擊「AI 風格轉換」展開選項，選擇想要的藝術風格
- **圖像增強**：一鍵提升圖像品質，自動調整對比度和飽和度
- **去背景**：智能識別主體，移除背景創建透明效果

## 🛠️ 技術特點

### 前端技術
- **純 HTML/CSS/JavaScript** - 無需額外依賴或框架
- **Canvas API** - 使用原生 Canvas 進行圖像處理
- **響應式設計** - 適配各種螢幕尺寸
- **現代化 UI** - 仿 Photoshop 的專業深色主題

### 核心算法
- **洪水填充算法** - 實現油漆桶智能填充功能
- **像素級操作** - 直接操作 ImageData 進行圖像處理
- **卷積濾鏡** - 實現模糊、銳化等濾鏡效果
- **歷史記錄系統** - 支援多步復原/重做操作

### 🤖 AI 算法
- **浮水印檢測** - 基於透明度和亮度異常檢測浮水印區域
- **內容感知修復** - 使用周圍像素信息智能填補缺失區域
- **風格轉換算法** - 色彩量化、對比度增強、色調調整等
- **邊緣檢測** - 用於背景分離和圖像增強
- **色彩空間轉換** - RGB/HSL 轉換實現精確色彩控制

### 架構設計
- **物件導向設計** - ImageEditor 類封裝所有功能
- **事件驅動** - 響應滑鼠和鍵盤事件
- **模組化功能** - 每個工具獨立實現
- **可擴展架構** - 易於添加新工具和功能
