# 每日新聞摘要生成器

一個基於 Python 的每日新聞摘要軟件，可以從 Google News 抓取指定主題的新聞，並生成摘要報告。

## 功能特點

### 核心功能
- 🌐 **Google News 整合**：從全球最大新聞平台抓取資訊
- 🖥️ **現代化圖形界面**：緊湊美觀的 GUI 設計
- 🎨 **動態背景**：根據時間自動變換背景顏色
- 🌓 **夜間模式**：保護眼睛的深色主題
- 🔍 **智能搜索**：頂部搜索欄快速搜索任意主題
- 📊 **時間排序**：支持最新/最舊排序
- 🔥 **熱門趨勢**：快速訪問熱門話題標籤

### 新聞展示
- 🖼️ **圖片顯示**：自動載入新聞縮圖
- 📡 **來源標識**：清晰顯示新聞來源
- 🕒 **時間戳記**：顯示抓取時間
- ⭐ **收藏功能**：收藏喜愛的新聞
- 🔗 **分享功能**：一鍵複製新聞鏈接
- 📊 **數據可視化**：圓餅圖顯示新聞類型分佈
- 🔍 **類別篩選**：點擊類別快速篩選相關新聞

### 其他功能
- 📝 **JSON 導出**：保存新聞為 JSON 格式
- 📧 **郵件通知**：支持郵件推送
- ⏰ **定時執行**：自動定時抓取
- 🌍 **多語言支持**：繁中、簡中、英文、日文
- ⚙️ **靈活配置**：可自定義各項設置

## 安裝

1. 克隆或下載此項目

2. 安裝依賴：
```bash
pip install -r requirements.txt
```

## 使用方法

### GUI 圖形界面（推薦）

運行圖形界面版本：
```bash
python news_gui.py
```

或使用快捷方式（Windows）：
```bash
start_gui.bat
```

#### 界面功能說明

**頂部區域**
- 🌤️ **天氣顯示**：顯示當前天氣和溫度（左側）
- 🔍 **搜索欄**：輸入任意主題並按 Enter 或點擊搜索（中間）
- 🕐 **實時時鐘**：顯示當前時間和日期（右側）
- 🌙 **模式切換**：切換日間/夜間模式

**導航區域**
- 🔥 **熱門標籤**：快速訪問熱門話題（AI、ChatGPT、台股等）
- 📋 **分類按鈕**：預設主題分類（科技、財經、體育、娛樂等）
- 🔍 **自定義搜索**：輸入自定義主題

**數據可視化**
- 📊 **圓餅圖**：顯示今日新聞類型分佈（科技、財經、體育等）
- 🎨 **語義化配色**：每個類別使用不同顏色
- 📈 **數據總結**：顯示新聞總數和類別數量
- 🔍 **點擊篩選**：點擊類別按鈕快速篩選相關新聞

**新聞卡片**
- 🖼️ **新聞圖片**：顯示新聞縮圖
- 📰 **標題摘要**：新聞標題和簡短摘要
- 📡 **來源信息**：顯示新聞來源媒體
- 🕒 **時間戳記**：顯示抓取時間
- 📖 **閱讀按鈕**：在瀏覽器中打開完整新聞
- 🔗 **分享按鈕**：複製新聞鏈接到剪貼板
- ⭐ **收藏按鈕**：收藏/取消收藏新聞

**底部操作**
- 💾 **保存**：將當前新聞保存為 JSON 文件
- 🗑️ **清空**：清空當前顯示的新聞
- ⭐ **收藏**：查看已收藏的新聞
- 📋 **顯示全部**：取消類別篩選，顯示所有新聞

### 命令行版本

運行主程序，手動輸入主題：
```bash
python news_scraper.py
```

### 定時執行

配置 `config.py` 中的設置，然後運行定時器：
```bash
python scheduler.py
```

程序將在每天指定時間自動抓取新聞。

### 郵件通知

1. 在 `config.py` 中配置郵件設置：
   - `sender_email`: 發件人郵箱
   - `sender_password`: 郵箱密碼或應用專用密碼
   - `recipient_email`: 收件人郵箱

2. 如果使用 Gmail，需要：
   - 啟用兩步驟驗證
   - 生成應用專用密碼：https://myaccount.google.com/apppasswords

3. 在代碼中取消註釋郵件發送部分

## 配置說明

### config.py

- `DEFAULT_LANGUAGE`: 設置默認語言（zh-TW, zh-CN, en-US, ja-JP）
- `MAX_ARTICLES`: 每個主題最多抓取的新聞數量
- `SCHEDULE_TIME`: 定時任務執行時間
- `TOPICS`: 關注的主題列表

### 支持的語言

- 繁體中文（台灣）
- 簡體中文（中國）
- English (US)
- 日本語

## 文件說明

- `news_scraper.py`: 核心爬蟲程序
- `email_notifier.py`: 郵件通知模塊
- `scheduler.py`: 定時任務調度器
- `config.py`: 配置文件
- `requirements.txt`: 依賴包列表

## 輸出格式

新聞數據保存為 JSON 格式，文件名格式：`{主題}_news_YYYY-MM-DD.json`

```json
[
    {
        "title": "新聞標題",
        "link": "新聞鏈接",
        "snippet": "新聞摘要",
        "scraped_at": "2025-12-08 10:30:00"
    }
]
```

## 注意事項

- 請遵守 Google News 的使用條款
- 不要過於頻繁地發送請求，避免被封鎖
- 使用郵件功能時，請妥善保管郵箱密碼
- 建議使用應用專用密碼而非主密碼
